# TDD E-Commerce Platform

A test-driven e-commerce system built with **JavaScript / Node.js** and **Jest**.

## How to Run Tests

npm install
npm test

## Requirements Covered

| Requirement | Topic |
|---|---|
| A | Product Model & Catalog |
| B | Shopping Cart |
| C | Inventory Reservation |
| D | Discount Rules |
| E | Checkout & Payment |
| F | Order History & Persistence |

## Key Concepts Practiced
- Red → Green → Refactor cycle
- Dependency injection
- Mocking external services
- Strategy pattern
- Repository pattern
